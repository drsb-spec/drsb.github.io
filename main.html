1. O que são tags?

As tags são como comandos ou marcadores que o HTML usa para dizer ao navegador o que cada parte do conteúdo é.
Elas sempre começam e terminam com colchetes angulares (< >).

2. O que são elementos?

Elementos HTML são os blocos fundamentais de uma página web.
Eles definem a estrutura e o conteúdo que o navegador exibe.
